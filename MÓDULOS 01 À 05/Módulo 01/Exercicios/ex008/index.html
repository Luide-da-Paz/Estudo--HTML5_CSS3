Principais Formatações

Negrito / Destaque

Para deixar um texto em negrito, você pode usar a tag <strong> ou <b>.

Para deixar o texto em itálico, você pode usar a tag <em> ou <i>.

Itálico / Ênfase

Para dar ênfase a um texto, você pode usar a tag <em> ou <i>.

lembrando que <em> e <i> têm o mesmo efeito visual, mas <em> é semanticamente mais correto para ênfase.

podemos criar também um texto marcado com a tag <mark> luide lindo.

texto grande e pequeno

Para deixar um texto maior, você pode usar a tag <h1> a <h6> para títulos, ou aumentar o tamanho da fonte com CSS.

Para deixar um texto menor, você pode usar a tag <small>.

texto excluído

Para excluir um texto, você pode usar a tag <del> ou <s>.

texto inserido

Para inserir um texto, você pode usar a tag <ins>.

texto sobrescrito

Para sobrescrever um texto, você pode usar a tag <sub>.

texto subrescrito

Para subescrever um texto, você pode usar a tag <sup>.
